# MediTrack.io

## Overview
MediTrack AI is an advanced telemedicine and health management dashboard designed to simplify healthcare for patients, especially the elderly. It leverages AI to interpret prescriptions, explain lab reports in plain English, and track medication adherence.

## Features
- **AI Prescription Reader**: Upload handwritten notes and get a digitized schedule.
- **Lab Report Explainer**: Translate complex medical jargon into simple health insights.
- **Medicine Adherence**: Daily checklists, reminders (WhatsApp/SMS), and tracking rings.
- **Family Sharing**: Keep loved ones informed with shared health dashboards.
- **Timeline**: A chronological history of all your medical events.
- **Dark Mode**: Fully supported accessible UI.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4 + Framer Motion
- **Icons**: Lucide React
- **Theme**: next-themes
- **Deploy**: Vercel/Netlify ready

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000)
